# Farewell Invitation — Department of Computer Science (Next.js)

A premium, responsive farewell invitation system for HOD Dr. S. Mary Vennila, built with Next.js.

## Features
- **Premium UI**: Shimmering gold aesthetics and smooth animations.
- **RSVP System**: Integrated with Google Sheets via Next.js Route Handlers.
- **Responsive**: Fully optimized for mobile and desktop.
- **Framework**: Next.js App Router.

## Deployment
Deployed on Vercel.
